# Code Snippets Catalog

A catalog of reusable, interview-ready code snippets organized by language. Each snippet is a self-contained implementation you can drop into a coding round: data structures, algorithms, and utility patterns written idiomatically for the host language.

Use this as a quick-reference library when practicing. The goal is not to memorize verbatim, but to internalize the shape and edge cases so you can reproduce a clean version under time pressure.

## Conventions

- Each snippet lives in its own file named after the structure or algorithm (e.g. `min-heap.md`, `union-find.md`).
- Snippets include the implementation, a one-line complexity summary, and a short usage example where helpful.
- Prefer generic / parameterized versions over hardcoded ones so they transfer across problems.
- Cross-link from the relevant topic primer (e.g. [[../07-advanced-data-structures/index|Advanced Data Structures]]) when the snippet is the canonical reference for a concept.
