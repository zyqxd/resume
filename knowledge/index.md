# Interview Prep Knowledge Bank

A structured knowledge base for Staff-level technical interviews, organized into two complementary sections. Built for the 2025-2026 interview landscape where AI/LLM fluency is expected alongside traditional distributed systems and coding fundamentals.

---

## Sections

### [[technical-and-coding/index|Technical & Coding]]

Data structures & algorithms, concurrency, OOP design, API design, testing, and AI/LLM integration. Each topic includes a primer with key concepts and hands-on exercises (timed code reviews and implementation problems in Ruby).

**6 topics, 8 exercises** — ~15-20 hours of focused study material.

### [[system-design/index|System Design]]

Scaling reads/writes, real-time systems, async processing, fault tolerance, databases, distributed systems fundamentals, API gateways, and ML/AI infrastructure. Each topic covers what, why, and trade-offs with decision frameworks.

**9 topics, 3 end-to-end design exercises** — ~20-25 hours of focused study material.

---

## How to Use

1. **Read the section index first** — each has a study priority guide
2. **Do exercises timed** — staff interviews are 45-60 minutes under pressure
3. **Follow the cross-links** — topics connect across sections (e.g., API Design in coding links to API Gateway in system design)
4. **Review answer keys after attempting** — don't peek early
